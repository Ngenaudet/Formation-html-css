HTML - les balises

  • <!DOCTYPE html>

    Cette balise déclare la version d'HTML que la page utilise. est utilisé pour indiquer que la page utilise HTML5, la dernière version d'HTML

  • <html>

    La balise est la racine de votre document HTML. Tout le contenu de la page est inclus entre les balises et

  • <head>

    La balise contient des éléments qui fournissent des informations sur la page, mais qui ne sont pas affichés directement à l'utilisateur. Cela peut inclure le titre de la page, des liens vers des feuilles de style CSS, des scripts JavaScript, etc.

  • <body>

    La balise englobe tout le contenu visible de la page, y compris le texte, les images, les liens, etc. C'est là que vous mettez le contenu principal de votre page.

  • <h1> <h2> <h3> <h4>

    Les balises <h1 > à <h6> sont utilisées pour définir les titres ou en-têtes de différentes tailles. <h1> est le titre principal, <h2> est un sous-titre, et ainsi de suite. Plus le numéro est élevé, plus le titre est petit.
